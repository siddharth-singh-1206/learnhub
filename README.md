# LearnHub

LearnHub is a responsive online learning platform built with React and Vite. It is designed to offer a clean and interactive environment for students to browse courses, view pricing models, read reviews, and get in touch with the support team.

---

## Features

- Responsive user interface
- React Router based navigation
- Reusable React components
- Dynamic rendering using map()
- FAQ Accordion
- Contact form with validation
- About page with team section
- Course listing page
- Pricing plans
- Testimonials section

## Getting Started

Follow these instructions to set up and run the project locally.

### Clone the repository

```bash
git clone <repository-url>
```

### Navigate to the project

```bash
cd learnhub
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

The application will run locally at:

```
http://localhost:5173
```

---

## Build for Production

To create an optimized production build of the application, run:

```bash
npm run build
```

This generates optimized static files in the `dist` folder ready to be deployed.

---

## Preview Production Build

To preview the production build locally before deploying, run:

```bash
npm run preview
```

This starts a local server pointing to the production bundle.
